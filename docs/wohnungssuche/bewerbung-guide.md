# Wohnungsbewerbung — Guide & Rules

Reference for drafting apartment-application messages for Zaid Alakad. Read this before crafting any application. This is currently a manual, assisted workflow (Claude drafts, Zaid sends). It may later become an automation.

## Goal

Given an apartment listing (a link or pasted text), produce a **ready-to-copy-paste German application message** tailored to that listing. Zaid pastes it into the portal / email and attaches his Bewerbermappe PDF.

## Hard rules (do / don't)

- **DRAFT ONLY. Never send anything.** Do not submit contact forms, send emails, or contact landlords. Claude produces text; Zaid sends it himself.
- **Never invent facts.** Only use the applicant facts below and details genuinely present in the listing. If a detail isn't in the listing, don't claim it.
- **No dashes inside sentences.** Never use `–` or `—` within a sentence. Always use a comma (or restructure) instead. Em/en dashes read as machine-written.
  - Exceptions that are fine: bullet-list markers (`– item`), and date/number spans like `07/2024–07/2026`.
- **Match the applicant's own style** from his Mappe: factual, confident, warm but not gushing.

## Per-listing fill-ins

For each listing, gather from the link/text:
1. **Contact name** → salutation. `Sehr geehrte Frau X,` / `Sehr geehrter Herr Y,`. If none, `Sehr geehrte Damen und Herren,`.
2. **Street / address** → used in the opening line and (optional) subject.
3. **One or two genuine listing details** → the personalized hook (Balkon, EBK, Altbau, Stadtteil, ruhige Lage, etc.).
4. Note **von privat vs. Makler**, and any mismatch with Zaid's wishes (move-in date, pets, etc.) worth flagging.

## Preferred structure (current default)

Zaid approved this richer structure (2026-07-12). Order:

1. **Optional subject line** `Betreff: Bewerbung um Ihre [Objekt] in der [Straße]` — delete for portals with their own subject field.
2. **Personalized hook** referencing 1–2 real listing features, ending on why it fits a long-term home.
3. **Who I am:** age, Nichtraucher, Einpersonenhaushalt, keine Haustiere; job + relocation reason; stable long-term tenant since 2022.
4. **"Sicherheit" bullet block** — the strongest selling point, up front: income, Rücklagen, SCHUFA, 2-year clean payment record.
5. **Bewerbermappe** mentioned as one attached PDF (contents in parentheses).
6. **Logistics:** Einzug 01.10.2026 (früher möglich), Besichtigung kurzfristig aus Hannover, phone number.
7. **Warm close** + `Mit freundlichen Grüßen` + name.

Keep the simpler original one-block version available as a fallback if a listing wants something shorter.
